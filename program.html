Fortran - 1957
Created by: John Backus
 
Fortran is the oldest programming language that is still in use. It was originally created for numerical computations and is primarily used in the field of science. Fortran 77 is an improved version of Fortran, as it introduced more control structures and enhanced support for character data. Fortran 90, on the other hand, now includes pointers and is much more reliable.
Cobol - 1959
Created by: Grace Murray Hopper
 
It was created because it's somewhat easier; it's easier to write and easier to read. The use of COBOL has decreased. Despite the reduced usage of COBOL, the Dutch government was still using COBOL as of 2018. In the past, COBOL was also prevalent on many Digital computers like VAX and Alpha. The group that had developed COBOL introduced COBOL-61, the new version, and when COBOL-61 was released, its usage also increased. COBOL-74 is an improved version and has a collaboration with ANSI. COBOL-2002 in this version, it is possible to use a multi-paradigm programming language.
C - 1972
Created by: Dennis Rithcie
 
The designer Dennis Ritchie also found success with Unix. Today, there are many programming languages that have their origins in C, such as C# and C++. Many programmers also enjoy working with C. As a result, C is one of the largest programming languages in the world.
Python - 1991
Created by: Guido van Rossum
 
Python is a widely-used programming language, particularly for data and games. Python employs duck typing, allowing you to adapt objects with new methods. It has a readable syntax, making it more enjoyable to read and write code. Python is also sometimes used for automating tasks.
PHP - 1994
Created by: Rasmus lerdorf
 
PHP's original name was "Personal Home Page Tools. Now, we simply call it "Hypertext Preprocessor" or "PHP." PHP is used worldwide.
Java - 1995
Created by: Oracle Corporation
 
With Java, you can create software for various platforms, using the "WORA" principle, which stands for "Write Once, Run Anywhere." Java allows you to develop services and applications.
Javascript - 1995
Created by: Brandon Eich
 
JavaScript is often regarded as the most widely used and best programming language. It is versatile and can be used to create web pages, web forms, web applications, and browser games. JavaScript was created by Brandon Eich.
HTML & CSS - 1996
Created by: Tim berners-Lee.
 
HTML and CSS are languages used for websites. HTML is used for creating websites, while CSS is used for designing their appearance. The latest versions are HTML5 and CSS3. Tim Berners-Lee played a significant role in their development.
